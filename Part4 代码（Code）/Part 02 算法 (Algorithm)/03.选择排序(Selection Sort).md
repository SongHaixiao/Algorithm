# 选择排序（Selection Sort）

## Code

### Java

```java
// java code

// 选择排序，a表示数组，n表示数组大小
public void selectionSort(int[] a, int n) {
	if (n <= 1) return;

    // 查找有序区的最小值直到末尾位置
    for(int i=0; i<n; i++)
    {
        int min=i;

        // 找出"a[i+1] ... a[n]"之间的最小元素
        // 并存储其 index 给 min。
        for(int j=i+1; j<n; j++)
        {
            if(a[j] < a[min])
                min=j;
        }

        // 若min!=i，则交换 a[i] 和 a[min]。
        // 交换之后，保证了a[0] ... a[i] 之间的元素是有序的。
        if(min != i){
            int temp = a[i];
            a[i] = a[min];
            a[min] = a[i];
        }
    }
}
```

### Python

```python
# python code

# selection sort, a - array, n - size of a
def selection_sort(a, n):
    if n <= 1:
        return
    
    # find the min until the end
    for i in range(n):
        min = i			# assume the i element is min, store it's index
        
        # find the min element among a[i+1]...a[n]
        # and store its index into min
        for j in range(i + 1, n):
            if a[j] < a[min]:
                min = j
        # if min != i, sawp a[i] and a[min]
        # then it will be stored among a[0]...a[i].
        if min != i:
            a[min], a[i] = a[i], a[min]
```
